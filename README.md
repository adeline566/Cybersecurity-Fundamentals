# Cybersecurity-Fundamentals


This repository contains a collection of hands-on cybersecurity labs covering fundamental security concepts and practical techniques. Topics include Linux administration, network traffic analysis, packet sniffing, ethical hacking, password cracking, steganography, and defensive security practices. Through these exercises, I gained experience working with industry-relevant tools and environments while developing skills in system security, network analysis, vulnerability assessment, and security operations. Each project includes documentation, screenshots, and explanations of the methodologies used, key findings, and lessons learned.

### Topics Covered
Linux Fundamentals
Network Traffic Analysis
Packet Sniffing and Monitoring
Ethical Hacking and Penetration Testing
Password Cracking
Steganography and Data Hiding
Offensive and Defensive Security Techniques
Security Operations Fundamentals

### Disclaimer

All activities documented in this repository were performed in controlled lab environments for educational purposes only. The techniques demonstrated are intended to improve cybersecurity knowledge and should only be used on systems where explicit authorization has been granted.
